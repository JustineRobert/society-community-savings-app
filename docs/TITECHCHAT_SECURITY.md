# TITechChat Security

## Principles
- JWT authentication on all endpoints
- Role‑based access control (RBAC)
- Audit‑safe soft deletion (no hard deletes)
- Attachment validation (type, size, ownership)
- Input sanitization against script injection
- Rate limiting middleware support
- Secure socket authentication and room isolation

## Compliance
- All actions logged in audit trail
- Export functions restricted to authorized roles
- Moderation hooks for abuse reporting
