# TITechChat Overview

TITechChat is the in‑app communication module for the TITech Community Capital platform. It enables secure, auditable, role‑based messaging across groups, admins, members, and support staff.

## Scope
- Group discussions
- Direct member ↔ admin/support messaging
- Loan/savings/transaction linked threads
- Helpdesk/support conversations
- Admin announcements and compliance functions

## Key Features
- Conversation creation and listing
- Message sending, editing, soft deletion
- Read receipts and typing indicators
- Attachments with secure validation
- Role‑based access enforcement
- Audit logging and export support
