# TITechChat Architecture

## Backend
- **Models**: Conversation, Message
- **Controllers**: chatController.js
- **Services**: chatService.js
- **Routes**: chat.js
- **Validation**: chatValidator.js
- **Realtime**: chatSocket.js

## Frontend
- **Pages**: TITechChat, ConversationList, ConversationDetail, SupportChat, LinkedThread
- **Components**: MessagePanel, Composer, ConversationList, etc.
- **State Management**: chatSlice.js
- **API Integration**: chatApi.js

## Cross‑Module Integration
- Notifications
- Loans, savings, transactions
- Support/helpdesk
- Audit logging
